## 🎯 Game Mechanics

### Level Progression
- Each level increases in:
  - Maze size
  - Number of obstacles
  - Number of traps
  - Complexity of key-lock puzzles
  - Time pressure

### Scoring System
- Base points for level completion
- Time bonus for quick completion
- Cumulative score tracking
- High score persistence

## 🔧 Technical Details

### Classes
- `PathfindingGame.java`: Main game controller and UI
- `GamePanel.java`: Game rendering and logic
- `Cell.java`: Individual maze cell properties

### Dependencies
- Java Swing for GUI
- Java AWT for graphics
- Java Collections Framework
